Odoo 11: CRM issue Module

This module is used to create project issue form lead.


How to use

You can find create issue button on the form of lead so if you want to create project issue from that lead you have to just click on that button.

Wizard for enter extra details releted to project issue

When you click on create issue button wizard will be open where you have to enter details of project ,dedline of the issue and assigned person details.

On lead form you can See Number of issue created from that lead .


Here you can see the detailed information of the project issue created from the lead.
